# YOURLS-sk_SK
Slovenský preklad YOURLS
**(Slovak YOURLS translation)**


## Aktuálna verzia

YOURLS **1.7**


## Použitie ?

1. Otvorte `config.php` a upravte riadok:
```
define( 'YOURLS_LANG', 'fr_FR' );
```
2. Následne prejdite do `user/languages`, a tam nahrajte súbory `sk_SK.po` a `sk_SK.mo`
